#+TITLE: spirw.sh

* About

Full duplex Read and Write command for Raspberry Pi SPI Interface implemented in shell script.

* Usage

#+BEGIN_SRC 
{
        echo 10100000
        yes  00000000   |
        head -n 10
}                       |
spirw			|
bin2hex
#+END_SRC

The easy way to debug SPI interface is 
shorting GPIO 10(MOSI) and GPIO 9(MISO) by jumper pin.
